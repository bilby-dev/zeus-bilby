# zeus-bilby

Plugin for using zeus with bilby.

This plugin exposes the `zeus` sampler via the `bilby.samplers` entry point.
Once installed, you can select it in `bilby.run_sampler` using `sampler='zeus'`.

## Installation

The package can be install using pip

```
pip install zeus-bilby
```

or conda

```
conda install conda-forge:zeus-bilby
```
